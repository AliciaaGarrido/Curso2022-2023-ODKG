Application Requirements

Application Name: Health Finder

Objectives

The application will allow users to search for health centers and public defibrillators. In the case of health centers, the user will be able to filter according to the desired specialty of the health center. And in the case of public defibrillators it will perform a quick search for the nearest one using the current location of the user.

Application description

The application will be divided in four screens:

  1. Main page: the page will provide users with the options to quick search for health centers or public defibrillators
  2. Defibrillators finder: after selecting the emergency option on the main page, the user will be redirected to this page, where using its location, it will display the nearest public defibrillator available.
  3. Health Center finder: the page will allow the user to search by location, either by using his/her current location or entering the zip code, and filter by the desired specialty of the ones viable in the list
  4. Health Center results: the page will display a map with the nearest health centers according to the locations and filters selected by the user
