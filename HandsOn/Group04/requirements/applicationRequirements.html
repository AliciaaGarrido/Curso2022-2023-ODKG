Application requirements

Idea

Buscador de fuentes o aseos públicos de Madrid ubicados en un barrio/distrito determinado.

User interface

Para seleccionar tanto el distrito como el barrio, utilizaremos desplegables. Para elegir entre fuentes o aseos, utilizaremos checkboxes. Tras darle al botón de "Búsqueda", la interfaz mostrará una lista con los elementos buscados.
