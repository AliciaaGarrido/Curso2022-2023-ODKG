This application will require that the user has to sign in before using the application. The user can sign in either using Email, Twitter or Facebook. To ensure new users, the application will provide functionality for creating a new account. After a successful login the application will provide a map of available bikes, which users can utilize to locate a nearby bike. Added functionality which will be applied are statistics about capacity, distance, available bikes and walking time. To improve the service provided, the application will also provide statistics from the rental process. These statistics can be used to anticipate how efficient the bike rental process is, which can in turn can be summarized in a report for the bike rental company. Reports for specific stations can indicate which stations have potential for improvement. This application will require data about: - Bike stations: identifier, name, capacity, operational status, available/non-available terminals, available/non-available bikes, geographical coordinates. - Bike: type e.g mechanical bike, electrical bike, available/non-available.
