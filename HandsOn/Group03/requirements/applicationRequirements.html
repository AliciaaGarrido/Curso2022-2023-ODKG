List of requirements to fulfill:

  • R1. As a user of the application I want to be able to find the schools in a neighborhood to find information about the school offerings in my neighborhood.
  • R2. As a user of the application I want to be able to find the schools in a district to find information about the school offerings in my district.
  • R3. As a user of the application I want to know the coordinates of a specific school in order to locate it.
  • R4. As a user of the application I want to know the data of a specific school (floor, number, street or directions) to be able to locate it physically.
  • R5. As a user of the application I want to be able to know the nearest form of transportation to a school in order to know how to get there.

Description of user interface:

This application consists of a search engine for public schools. Searches can be filtered according to various parameters (coordinates, district, neighborhood) as well as offering visualization on a map of the schools. Finally, this search engine will allow us to obtain information associated with each of the specific schools (nearest form of transport, description, directions, etc).
