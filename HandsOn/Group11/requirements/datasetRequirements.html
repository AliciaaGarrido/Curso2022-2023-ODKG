DATASET REQUIREMENTS

1. Data in smart city domain

The data comes from Madrid City Council. It includes the data of commemorative plaques in multiple building facades in Madrid.

The idea is to show on the facades of the houses of the city the relationship of the events that took place with historical value, of the relevant people who lived in those properties and also to highlight the importance of certain unique buildings.

Inside the Smart City domain, it is related to Smart People, focusing on the culture and entertaiment area.


2. Data available as a CSV file

The dataset is available in CSVfile. Dataset available here.


3. Data have an open license so it can be published

    We can use the data, as long as we attribute public use of the database, keeps the same license and keeps it open for others to use.

    This license satifies the third requirement so it can be published.


    4. Data can be easily linked with generic real-world entities

    The dataset contains full addresses and latitude-longitude coordinates of the plaques which can be linked with generic real-world entities like locations and municipalities.

    In the dataset we can also see Key Words for each commemorative plaque. These keywords may include the time period, the profession of the historical character and other concepts that allow to easily link the data with generic real-world entities.
