Application Requirements

Description

Our application will allow users to find cigar stores which are authorised to sell public transport cards in Madrid.

User Interface

The application will detect the user's location and display a map with the nearest cigar stores.

  • User location (black arrow) and the nearest cigar stores (blue dots)

    • The user will be able to select their desired cigar store and the application will calculate the route to the desired one.

  • Route to the selected cigar store
