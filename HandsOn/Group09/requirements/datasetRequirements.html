Dataset description

The dataset contains the ubications of cigar stores of the province of Madrid that can be used to request or recharge the public transport pass. The source is here.

Dataset license

The dataset comes with a LDA license (Licencia de Datos Abiertos) which allows the following (among others):

  • Download the data for free.
  • Share the data under the same license.
The license also has the following restrictions (among others):
  • It is not allowed to alter the data in order to modify the represented knowledge.
  • The CRTM must be cited as the source of the data.
  • Any application that utilizes this data must include the text "Powered by CRTM" and a link to http://www.crtm.es/.
  • It is not allowed to alter/delete the metadata regarding the updating date and license.
More information about the license can be found here.

Dataset requirements

R1: Data are in the smart cities domain.

The owner of the dataset is the municipality of Madrid. More precisely the entity Consorcio Regional de Transportes de Madrid (CRTM) so it is in one of the selected city domains.

R2: Data are available as a CSV file.

The dataset is available in CSV format as it is displayed in its webpage: Link to page

R3: Data have an open license so it can be published.

The dataset has an open license so people can use the data in derived forms. This license can be found here: Link to page

R4: Data can be easily linked with generic real-world entities (e.g., locations).

The dataset contains full addresses and latitude-longitude coordinates of cigar stores which can be linked with generic entities such as municipalities or provinces.

R5: Documentation exists for the data (optional).

This requirement is also satisfied since the source website provides information about the data type of each column of the dataset.

R6: Data can come from multiple data sources (optional).

The only source is the CRTM (Consorcio Regional de Transportes de Madrid), so this requirement is not satisfied.
