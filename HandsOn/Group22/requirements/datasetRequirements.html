Dataset 1: Inventario de zonas verdes

R1. Data are in the smart cities domain

One of the major domains of smart cities is the environment, and this dataset is exactly about that, a set of Green Zones (parks, gardens and forest areas) in Madrid.

R2. Data are available as a CSV file

The URL in which the dataset was found (https://datos.madrid.es/portal/site/egob/menuitem.c05c1f754a33a9fbe4b2e4b284f1a5a0/?vgnextoid=79c6adbb460dd510VgnVCM2000001f4a900aRCRD&vgnextchannel=374512b9ace9f310VgnVCM100000171f5a0aRCRD&vgnextfmt=default) has the dataset available in csv and xlsx formats.

R3. Data have an open license so it can be published

It is open data, its license allows the reuse of the documents and data subject to them for commercial and non-commercial purposes.

R4. Data can be easily linked with generic real-world entities (e.g., locations)

The dataset satisfies the requirement as it contains data about the green spaces of the districts of Madrid, which can have the role of entities when linking the dataset with others.

This selected dataset1 can be linked with the selected dataset2 (and with other external datasets) by using the entity of the districts (Distrito attribute in dataset1), that is also available in dataset2 (Nombre_distrito attribute in dataset2).

R5. Documentation exists for the data (optional)

There is documentation for the dataset referring to the structure of the dataset, it gives additional information about the definition of the columns in the dataset.

R6. Data can come from multiple data sources (optional)

It is only known that the data comes from the ‘Inventario de Vías Públicas y Zonas Verde’ of Madrid' s City Council.
