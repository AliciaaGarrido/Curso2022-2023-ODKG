Dataset 1: Red de Calidad del Aire. Datos del día en curso

  • R1. Data are in the smart cities domain

    • One of the major domains of smart cities is the air quality, and this dataset is exactly about that, a set of hourly data automatically measured at the stations of the Air Quality Network of the Community of Madrid, during the current day.

  • R2. Data are available as a CSV file

    • The URL in which the dataset was found (https://datos.comunidad.madrid/catalogo/dataset/calidad_aire_datos_dia) has the dataset available in csv, json and pdf formats.

  • R3. Data have an open license so it can be published

    • It is open data, with a ‘Creative Commons Attribution (CC BY 4.0)’ license.

  • R4. Data can be easily linked with generic real-world entities (e.g., locations)

    • The dataset satisfies the requirement as it contains data about provinces and municipalities of Madrid as entities.
    • For the provinces, there is only one, Madrid, identified as ‘28’ in the dataset.
    • For the municipalities, we have 24 of them (ALCALÁ DE HENARES, ALCOBENDAS, ALCORCÓN, ALGETE, ARANJUEZ, ARGANDA DEL REY, EL ATAZAR, COLMENAR VIEJO, COLLADO VILLALBA, COSLADA, FUENLABRADA, GETAFE, GUADALIX DE LA SIERRA, LEGANÉS, MAJADAHONDA, MÓSTOLES, ORUSCO DE TAJUÑA, PUERTO DE COTOS, RIVAS-VACIAMADRID, SAN MARTÍN DE VALDEIGLESIAS, TORREJÓN DE ARDOZ, VALDEMORO, VILLA DEL PRADO, VILLAREJO DE SALVANÉS).

  • R5. Documentation exists for the data (optional)

    • There is documentation for the dataset referring to the structure of the dataset, it gives additional information about the meaning of the different values that can be found, for example, in the municipalities column when the value ‘5’ is present, it refers to the Alcalá de Henares municipality.

    • The data comes from different stations of the Air Quality Network of the Community of Madrid.

Dataset 2: Smart Road: Parking occupancy

  • R1. Data are in the smart cities domain

    • This dataset is related to the smart road domain, and it shows the number of occupied and free parking spaces in the two zones “blue” and “yellow”. It also includes the detection of free parking by camera, but it is still in its test fase.

  • R2. Data are available as a CSV file

    • The dataset’s information can be obtained in https://data.europa.eu/data/datasets/100160-kanton-basel-stadt?locale=en. Here we can see it is possible to download the data in CSV and JSON formats.

  • R3. Data have an open license so it can be published

    • The license of this dataset is CC BY 3.0 CH, and it allows to:
      Share: copy and redistribute the material in any medium or format.
      Adapt: remix, transform and build upon the material for any purpose, even commercially

  • R4. Data can be easily linked with generic real-world entities (e.g., locations)

    • Data is related to parking occupancy of Switzerland. In this link the exact locations of the different sensors used can be found https://data.europa.eu/data/datasets/100160-kanton-basel-stadt?locale=en.

  • R5. Documentation exists for the data (optional)

    • The publisher of the dataset did not provide any documentation for the data. We can only find an explanation of the data contained and information related to the different sensors.

  • R6. Data can come from multiple data sources (optional)

    • There is one publisher for the dataset which is Amt für Mobilität, it belongs to the Canton of Zürich.
